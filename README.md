# Installation
```
git clone https://github.com/Trabbit0ne/spyder && cd spyder && chmod +x * && ./main.sh && echo 'alias spyder="cd ~/spyder/ && ./main.sh"' > ~/.bashrc && source ~/.bashrc
```
Then

``
spyder -u <username> -t <target(s)>
``
