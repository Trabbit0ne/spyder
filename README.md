# Installation
```
curl -sSL https://raw.githubusercontent.com/Trabbit0ne/spyder/refs/heads/main/main.sh -o $(dirname $(command -v bash))/spyder && chmod +x $(dirname $(command -v bash))/spyder
```
Then

``
spyder -u <username> -t <target(s)>
``
